> It is my intention to build a package set, and yes I will make it available. 
> I DID run make install - is there another step I am missing ?

In the spec file I use for building rpm packages I call:

	make buildroot=$RPM_BUILD_ROOT \
		bindlib=%{_lib} \
		lib_prefix=$RPM_BUILD_ROOT/usr/%{_lib} \
		doc_prefix=$RPM_BUILD_ROOT/%{_defaultdocdir} \
		man_prefix=$RPM_BUILD_ROOT/%{_mandir} \
	install

that's all. I think you didn't miss anything
